STD :: System_error

C Thư viện<system_error> 
lớp system_error;
 (kể từ C ++ 11)

std::system_errorlà loại ngoại lệ được ném bởi các chức năng thư viện khác nhau (thường là các chức năng giao diện với các cơ sở hệ điều hành, ví dụ: hàm tạo củastd::thread) khi ngoại lệ có liên quanstd::error_code, có thể được báo cáo.
