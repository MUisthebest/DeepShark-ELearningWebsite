Thư viện tiện ích

C ++ bao gồm một loạt các thư viện tiện ích cung cấp chức năng từbit-countingĐẾNpartial function application. Các thư viện này có thể được chia thành hai nhóm:

  • Thư viện hỗ trợ ngôn ngữ và
  • Thư viện đa năng.

Hỗ trợ ngôn ngữ

Thư viện hỗ trợ ngôn ngữ cung cấp các lớp và chức năng tương tác chặt chẽ với các tính năng ngôn ngữ và hỗ trợ các thành ngữ ngôn ngữ chung.

Type support

Các loại cơ bản (ví dụ:std::size_tThìstd::nullptr_t), Rtti (ví dụ:std::type_info), loại đặc điểm (ví dụ:std::is_integralThìstd::rank)

Program utilities

Chấm dứt (ví dụ:std::abortThìstd::atexit), môi trường (ví dụ:std::system), tín hiệu (ví dụ:std::raise)

Dynamic memory management

Con trỏ thông minh (ví dụ:std::shared_ptr), phân bổ (ví dụ:std::allocatorhoặcstd::pmr::memory_resource), Quản lý bộ nhớ kiểu C (ví dụ:std::malloc)

Error handling

Ngoại lệ (ví dụ:std::exceptionThìstd::terminate), xác nhận (ví dụ:assert)
