Từ khóa C ++: Lớp

Cách sử dụng

  • declaration of a class
  • declaration of a scoped enumeration type
(kể từ C ++ 11)
  • Trong a template declarationThìclasscó thể được sử dụng để giới thiệu type template parametersVà template template parameters
  • Nếu một hàm hoặc một biến tồn tại trong phạm vi với tên giống hệt với tên của loại lớp,classcó thể được chuẩn bị cho tên để định hướng, dẫn đến mộtelaborated type specifier

Ví dụ

lớp foo; // Tuyên bố về phía trước của một lớp học
