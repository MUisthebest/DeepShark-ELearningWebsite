Từ khóa C ++: Constexpr(kể từ C ++ 11)

Cách sử dụng

  • constexpr declaration specifier (kể từ C ++ 11)
  • constexpr if statement (C ++ 20)
  • Lambda-Declarator chỉ định rõ ràng lệnh gọi chức năng là hàm constexpr(C ++ 20)
