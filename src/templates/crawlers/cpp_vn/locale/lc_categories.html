Lc_all, lc_collate, lc_ctype, lc_monetary, lc_numeric, lc_time

C Thư viện<clocale> 
#Define lc_all / * Xác định triển khai * / /
 
#Define lc_collate / * Xác định triển khai * / /
 
#Define lc_ctype / * Xác định triển khai * / /
 
#define lc_monetary / * Xác định triển khai * / /
 
#Define lc_numeric / * Xác định triển khai * / /
 
#define lc_time / * đã xác định triển khai * / /

Mỗi hằng số macro trên mở rộng thành các biểu thức hằng số nguyên với các giá trị riêng biệt phù hợp để sử dụng làm đối số đầu tiên củastd::setlocale.

Đặc điểm kỹ thuật: Lưu ý: Đồng bộ hóa giữa các luồng tập trung vào việc ngăn chặn các chủng tộc dữ liệu (bằng cách thiết lập các mối quan hệ xảy ra trước khi) và xác định các tác dụng phụ nào có thể nhìn thấy được trong các điều kiện nào.
LC_ALL chọn toàn bộ địa phương c
LC_COLLATE chọn loại đối chiếu của địa phương C
LC_CTYPE Chọn loại phân loại ký tự của địa phương C
LC_MONETARY Chọn loại định dạng tiền tệ của địa phương C
LC_NUMERIC Chọn loại định dạng số của địa phương C
LC_TIME Chọn loại định dạng thời gian của địa phương C

Các hằng số macro bổ sung, với các tên bắt đầu bằngLC_theo sau là ít nhất một chữ cái viết hoa, có thể được xác định trong<clocale>. Ví dụ: đặc điểm kỹ thuật POSIX yêu cầuLC_MESSAGES(điều khiểnstd::perrorVàstd::strerror), ISO/IEC 30112: 2014 (2014 draft) Ngoài ra định nghĩaLC_IDENTIFICATIONThìLC_XLITERATEThìLC_NAMEThìLC_ADDRESSThìLC_TELEPHONEThìLC_PAPERThìLC_MEASUREMENT, VàLC_KEYBOARD, được hỗ trợ bởi thư viện GNU C (ngoại trừLC_XLITERATE).

#include <clocale>
